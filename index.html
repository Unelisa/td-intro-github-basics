What is Version Control

A system known as version control tracks changes made to a group of files or a piece of code over time.
It makes it possible for numerous contributors to work on the same project at once while keeping track of the changes made by each one.
Version control is mostly used in software development projects to keep track of changes made and to promote developer collaboration.

Git

Git is a distributed version control system that is frequently employed in the creation of software and other team-based initiatives.
At its core, Git is a technology that enables teamwork by enabling several
developers to work on the same codebase concurrently while tracking changes, recording modifications, and maintaining a history of changes.

Process of commiting to git

  • a new git repo using git init
  • use git add . to add all files changed
  • commit the changes, use - git commit -m "My Message"
